# Chef Claude 

A smart kitchen assistant that helps you turn whatever is in your fridge into a delicious meal. Built with React and powered by AI.

## About the Project
Chef Claude solves the "what should I cook?" dilemma. Simply enter the ingredients you have on hand, and the app uses the Hugging Face API to generate a custom recipe for you. It’s a great example of dynamic state management and handling asynchronous API requests in React.

## Tech Stack
* **Framework:** React
* **Styling:** CSS (Flexbox)
* **AI Engine:** Hugging Face API
* **Deployment:** Vercel
* Link https://chef-claude-tawny.vercel.app/
